# This is a simple RSVP web application.
To track invitees
### Features:

- "Invite" someone
- Check "confirmed" if the person responded
- Filtering
  - hide unresponded
  - show only unresponded
- Edit the invitee
- Remove the invitee
- Download the invitees and their responses
  - in CSV format
  - in TXT format

Should be mobile responsive.
Does **not** have session persistence.
